# ninja-platformer-game
Game project description
A 2D side-scrolling ninja platformer game developed using JavaScript and the p5.js library. In this game, the player controls a ninja character who must collect rotating stars, avoid enemies and lava canyons, jump across moving platforms, and reach the flagpole before the timer runs out.

The game features an evening sky environment with moving clouds, mountains, sound effects, score tracking, a lives system, and fireworks celebration upon completing the level.
